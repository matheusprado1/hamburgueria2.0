## Descrição

Este respositório contem a versão melhorada da hamburgueria que criei com React `https://github.com/matheusprado1/dev-burger`. Esta nova versão foi feita com TypeScript.

## Tecnologias

- TypeScript
- React.js
- Styled-components
- React-hook-form
- Zod
- React-icons
- React-toastify

